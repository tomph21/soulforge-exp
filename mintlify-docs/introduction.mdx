## Tool suite

SoulForge ships 36+ tools organized by capability:

### Code intelligence

| Tool | What it does |
|------|-------------|
| `read` | Read files — full, line ranges, or surgical symbol extraction by name |
| `navigate` | Definition, references, call hierarchy, implementations, type hierarchy |
| `analyze` | File diagnostics, unused symbols, type info, code actions |
| `rename_symbol` | Compiler-guaranteed rename across all files |
| `move_symbol` | Move to another file + update all importers |
| `refactor` | Extract function/variable, organize imports, format |

### Codebase analysis (zero LLM cost)

| Tool | What it does |
|------|-------------|
| `soul_grep` | Token-efficient search with count mode and repo map intercept |
| `soul_find` | Fuzzy file/symbol search, PageRank-ranked, signatures included |
| `soul_analyze` | Identifier frequency, unused exports, file profiles, duplication, packages |
| `soul_impact` | Dependency graph — dependents, cochanges, blast radius |

### Project management

| Tool | What it does |
|------|-------------|
| `project` | Auto-detected lint, format, test, build, typecheck across 23 ecosystems |
| `dispatch` | Parallel multi-agent execution with shared cache |
| `git` | Structured git operations with auto co-author tracking |
| `editor` | 18 Neovim actions — read, edit, navigate, LSP, format, and more |
